# Orange_Storm_Giga_4_Extruder_Printer_Profiles
Printer profiles, Generic Filament and Basic Process Profiles for the Elegoo Orange Storm Giga with the 4 Extruder Mod. Elegoo doesn't seem to have any profiles, all of the community ones I found wouldn't work for me. So I made my own. These ar very basic profiles and will need tweaked, but they should get you started.
